# if you want early build or want to compile here are the steps

Step1 : Install Packages
```bash
	yarn
```
Step2 : compile main jsx file
```bash
	yarn start:control
```
Step3 : Run it
```bash
	yarn start
```

Step 5 (Optional) : Compiling dist aka exe or deb

```bash
	yarn dist
```
